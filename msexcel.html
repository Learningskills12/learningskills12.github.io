📊 MS Excel কী?

Microsoft Excel হলো Microsoft-এর তৈরি একটি জনপ্রিয় Spreadsheet Software। এটি দিয়ে হিসাব-নিকাশ, ডেটা সংরক্ষণ, বিশ্লেষণ, চার্ট তৈরি এবং বিভিন্ন ধরনের রিপোর্ট তৈরি করা যায়।

📒 Workbook ও Worksheet

Workbook: Excel-এর সম্পূর্ণ ফাইলকে Workbook বলা হয়।

Worksheet: Workbook-এর প্রতিটি Sheet-কে Worksheet বলা হয়।

🔢 Row, Column ও Cell

  • Row: অনুভূমিক (Horizontal) সারি। সংখ্যা (1, 2, 3...) দিয়ে চিহ্নিত।
  • Column: উল্লম্ব (Vertical) সারি। অক্ষর (A, B, C...) দিয়ে চিহ্নিত।
  • Cell: Row ও Column-এর মিলিত স্থানকে Cell বলা হয়।

উদাহরণ: A1, B5, C10

✍️ Data Entry

Cell-এ ক্লিক করে সংখ্যা, লেখা বা Formula লিখে Enter চাপলেই Data সংরক্ষিত হয়।

💾 Save ও Open

Save: Ctrl + S

Open: Ctrl + O

New Workbook: Ctrl + N

💡 পরীক্ষার জন্য গুরুত্বপূর্ণ তথ্য

  • ✅ MS Excel একটি Spreadsheet Software।
  • ✅ Workbook-এর ভিতরে এক বা একাধিক Worksheet থাকে।
  • ✅ Row সংখ্যা দ্বারা চিহ্নিত হয়।
  • ✅ Column ইংরেজি অক্ষর দ্বারা চিহ্নিত হয়।
  • ✅ Cell হলো Row ও Column-এর মিলিত স্থান।
  • ✅ Ctrl + S = Save
  • ✅ Ctrl + O = Open
  • ✅ Ctrl + N = New Workbook

➕ গুরুত্বপূর্ণ Excel Formula

  • SUM: =SUM(A1:A10) → মোট যোগফল বের করে।
  • AVERAGE: =AVERAGE(A1:A10) → গড় বের করে।
  • MAX: =MAX(A1:A10) → সর্বোচ্চ মান বের করে।
  • MIN: =MIN(A1:A10) → সর্বনিম্ন মান বের করে।
  • COUNT: =COUNT(A1:A10) → সংখ্যাযুক্ত Cell গণনা করে।

📊 Chart

Chart ব্যবহার করে ডেটাকে গ্রাফ আকারে দেখানো যায়।

Insert → Charts

📋 Table

Excel-এ Table তৈরি করতে:

Insert → Table

Shortcut Key: Ctrl + T
