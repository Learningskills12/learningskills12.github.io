💾 Pen Drive কী?

Pen Drive বা USB Flash Drive হলো একটি ছোট, বহনযোগ্য (Portable) Storage Device, যা Flash Memory ব্যবহার করে তথ্য সংরক্ষণ ও স্থানান্তর করতে ব্যবহৃত হয়। এটি USB Port-এর মাধ্যমে Computer, Laptop, Smart TV এবং অন্যান্য Device-এর সাথে সংযুক্ত করা যায়।

🎯 Pen Drive-এর প্রধান কাজ

  • 💾 Data সংরক্ষণ করা।
  • 📂 এক Device থেকে অন্য Device-এ File Transfer করা।
  • 🖥️ Software ও Operating System বহন করা।
  • 📸 ছবি, ভিডিও ও Document Backup রাখা।
  • 🎵 Audio ও Video সহজে বহন করা।

📜 Pen Drive-এর ইতিহাস

২০০০ সালের দিকে USB Flash Drive বাজারে আসে। এটি Floppy Disk ও CD-এর আধুনিক বিকল্প হিসেবে জনপ্রিয় হয়ে ওঠে। বর্তমানে কয়েক GB থেকে ১ TB বা তারও বেশি ধারণক্ষমতার Pen Drive পাওয়া যায়।

⚙️ Pen Drive কীভাবে কাজ করে?

Pen Drive-এর ভিতরে Flash Memory Chip থাকে। USB Port-এর মাধ্যমে এটি Computer-এর সাথে সংযুক্ত হলে Operating System এটিকে একটি Storage Device হিসেবে শনাক্ত করে। এরপর ব্যবহারকারী সহজেই File Copy, Paste, Delete বা Transfer করতে পারেন।

🛠️ Pen Drive কীভাবে ব্যবহার করবেন?

  1. USB Port-এ Pen Drive সংযুক্ত করুন।
  2. Computer এটি শনাক্ত করার জন্য কয়েক সেকেন্ড অপেক্ষা করুন।
  3. File Explorer খুলে Pen Drive নির্বাচন করুন।
  4. প্রয়োজনীয় File Copy, Paste বা Delete করুন।
  5. কাজ শেষে নিরাপদভাবে Eject করে Pen Drive খুলুন।

📜 Pen Drive-এর বিস্তারিত ইতিহাস

Pen Drive বা USB Flash Drive ২০০০ সালের দিকে বাজারে আসে। এটি Floppy Disk, Zip Drive এবং CD-এর তুলনায় ছোট, দ্রুত এবং বহনযোগ্য হওয়ায় খুব দ্রুত জনপ্রিয় হয়ে ওঠে। বর্তমানে USB 3.2 ও USB Type-C Pen Drive-এর মাধ্যমে অনেক দ্রুত গতিতে Data Transfer করা যায় এবং উচ্চ ধারণক্ষমতার Pen Drive-ও সহজলভ্য।

❓ FAQ (প্রশ্ন ও উত্তর)

১. Pen Drive কী?

Pen Drive হলো একটি Portable USB Flash Storage Device, যা Data সংরক্ষণ ও স্থানান্তরের জন্য ব্যবহৃত হয়।


২. Pen Drive কোন ধরনের Memory ব্যবহার করে?

Flash Memory ব্যবহার করে।


৩. Pen Drive কি বারবার ব্যবহার করা যায়?

হ্যাঁ। এতে বহুবার Data লেখা, মুছে ফেলা এবং পুনরায় সংরক্ষণ করা যায়।


৪. Pen Drive কোথায় ব্যবহার করা হয়?

Computer, Laptop, Smart TV, OTG সমর্থিত Smartphone, Printer এবং অন্যান্য USB Device-এ।


৫. Pen Drive ব্যবহারের পর Eject করা কেন দরকার?

Data নিরাপদ রাখার জন্য এবং File Corruption এড়ানোর জন্য।
