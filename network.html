🌐 Computer Network কী?

Computer Network হলো দুই বা ততোধিক কম্পিউটার বা ডিভাইসকে সংযুক্ত করে তথ্য (Data) ও বিভিন্ন রিসোর্স (যেমন Printer, Internet, File) ভাগাভাগি করার ব্যবস্থা।

🖥️ LAN (Local Area Network)

LAN (Local Area Network) হলো ছোট একটি নির্দিষ্ট এলাকার নেটওয়ার্ক, যেমন একটি বাড়ি, স্কুল, অফিস বা কম্পিউটার ল্যাব।

উদাহরণ: স্কুলের কম্পিউটার ল্যাব।

🏙️ MAN (Metropolitan Area Network)

MAN (Metropolitan Area Network) একটি শহর বা বড় এলাকার মধ্যে একাধিক LAN-কে সংযুক্ত করে।

উদাহরণ: একটি শহরের বিভিন্ন সরকারি অফিসের নেটওয়ার্ক।

🌍 WAN (Wide Area Network)

WAN (Wide Area Network) অনেক বড় ভৌগোলিক এলাকাজুড়ে বিস্তৃত নেটওয়ার্ক। Internet হলো সবচেয়ে বড় WAN-এর উদাহরণ।

📊 LAN, MAN ও WAN-এর তুলনা

নেটওয়ার্ক ব্যাপ্তি উদাহরণ
LAN ছোট এলাকা স্কুল, অফিস
MAN একটি শহর City Network
WAN দেশ বা বিশ্ব Internet

📡 Network Devices

  • 📶 Modem: Internet সংযোগ স্থাপন করে।
  • 📡 Router: Internet একাধিক ডিভাইসে ভাগ করে।
  • 🔀 Switch: একই Network-এর ডিভাইসগুলোকে সংযুক্ত করে।
  • 🔗 Hub: একাধিক কম্পিউটার সংযোগ করে (বর্তমানে কম ব্যবহৃত)।

💡 পরীক্ষার জন্য গুরুত্বপূর্ণ তথ্য

  • ✅ LAN = Local Area Network
  • ✅ MAN = Metropolitan Area Network
  • ✅ WAN = Wide Area Network
  • ✅ Internet হলো সবচেয়ে বড় WAN।
  • ✅ Router Internet ভাগ করে।
  • ✅ Modem Internet সংযোগ প্রদান করে।

❓ Computer Network MCQ

  1. Computer Network কী?
    A) একটি সফটওয়্যার
    B) দুই বা ততোধিক কম্পিউটার সংযোগের ব্যবস্থা
    C) একটি অপারেটিং সিস্টেম
    D) একটি ব্রাউজার
    উত্তর: B) দুই বা ততোধিক কম্পিউটার সংযোগের ব্যবস্থা

  2. LAN-এর পূর্ণরূপ কী?
    A) Local Area Network
    B) Large Area Network
    C) Long Area Network
    D) Link Area Network
    উত্তর: A) Local Area Network

  3. MAN-এর পূর্ণরূপ কী?
    A) Main Area Network
    B) Metropolitan Area Network
    C) Multiple Area Network
    D) Mega Area Network
    উত্তর: B) Metropolitan Area Network

  4. WAN-এর পূর্ণরূপ কী?
    A) Wide Area Network
    B) Web Area Network
    C) Wireless Area Network
    D) World Area Network
    উত্তর: A) Wide Area Network

  5. Internet কোন ধরনের Network?
    A) LAN
    B) MAN
    C) WAN
    D) PAN
    উত্তর: C) WAN

  6. Router-এর কাজ কী?
    A) Print করা
    B) Internet একাধিক ডিভাইসে ভাগ করা
    C) Scan করা
    D) গান শোনানো
    উত্তর: B) Internet একাধিক ডিভাইসে ভাগ করা

  7. Modem-এর কাজ কী?
    A) ছবি আঁকা
    B) Internet সংযোগ প্রদান করা
    C) Document লেখা
    D) Video চালানো
    উত্তর: B) Internet সংযোগ প্রদান করা

  8. Switch কোথায় বেশি ব্যবহৃত হয়?
    A) একই Network-এর ডিভাইস সংযোগে
    B) Printer-এ
    C) Monitor-এ
    D) Keyboard-এ
    উত্তর: A) একই Network-এর ডিভাইস সংযোগে

  9. Hub কী?
    A) একটি Browser
    B) একটি Network Device
    C) একটি Software
    D) একটি Operating System
    উত্তর: B) একটি Network Device

  10. নিচের কোনটি সবচেয়ে বড় Network?
    A) LAN
    B) MAN
    C) WAN
    D) Internet
    উত্তর: D) Internet
