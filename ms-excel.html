📊 MS Excel কী?

MS Excel (Microsoft Excel) হলো Microsoft Corporation-এর তৈরি একটি জনপ্রিয় Spreadsheet Software। এটি ব্যবহার করে সংখ্যা, তথ্য (Data), হিসাব-নিকাশ, টেবিল, চার্ট, রিপোর্ট এবং বিশ্লেষণ (Analysis) করা যায়।

🎯 MS Excel-এর প্রধান কাজ

  • ➕ হিসাব-নিকাশ (Calculation) করা।
  • 📋 Data সংরক্ষণ ও পরিচালনা করা।
  • 📊 Chart ও Graph তৈরি করা।
  • 📈 Data Analysis করা।
  • 🧾 Salary Sheet, Marksheet ও Budget তৈরি করা।
  • 📄 রিপোর্ট তৈরি করা।
  • 🖨️ Spreadsheet Print করা।

📜 MS Excel-এর ইতিহাস

Microsoft প্রথম ১৯৮৫ সালে MS Excel প্রকাশ করে। বর্তমানে এটি Microsoft 365-এর অংশ এবং বিশ্বের অন্যতম জনপ্রিয় Spreadsheet Software হিসেবে ব্যবহৃত হয়।

🧮 MS Excel-এর গুরুত্বপূর্ণ Formula

Formula কাজ
=SUM(A1:A10) সংখ্যার যোগফল বের করে
=AVERAGE(A1:A10) গড় বের করে
=MAX(A1:A10) সবচেয়ে বড় সংখ্যা বের করে
=MIN(A1:A10) সবচেয়ে ছোট সংখ্যা বের করে
=COUNT(A1:A10) সংখ্যাযুক্ত Cell গণনা করে
=IF(A1>=33,"Pass","Fail") শর্ত অনুযায়ী ফলাফল দেখায়
=TODAY() বর্তমান তারিখ দেখায়
=NOW() বর্তমান তারিখ ও সময় দেখায়

📜 MS Excel-এর বিস্তারিত ইতিহাস

Microsoft Corporation ১৯৮৫ সালে প্রথম MS Excel প্রকাশ করে। প্রথমে এটি Apple Macintosh-এর জন্য তৈরি করা হয় এবং পরে Windows-এর জন্য প্রকাশিত হয়। সময়ের সাথে Excel-এ Formula, Function, Chart, Pivot Table, Power Query, Macro (VBA), Data Analysis এবং Cloud Collaboration-এর মতো উন্নত সুবিধা যুক্ত হয়েছে। বর্তমানে Microsoft 365-এর অংশ হিসেবে Excel Online-ও ব্যবহার করা যায়।

১. MS Excel কী?

২. MS Excel কে তৈরি করেছে?

৩. MS Excel-এর Default File Extension কী?

৪. Excel-এ Formula কেন ব্যবহার করা হয়?

দ্রুত ও নির্ভুলভাবে হিসাব-নিকাশ করার জন্য Formula ব্যবহার করা হয়।


৫. Excel-এর সবচেয়ে জনপ্রিয় Function কোনগুলো?

SUM(), AVERAGE(), MAX(), MIN(), COUNT(), IF(), VLOOKUP(), XLOOKUP() (নতুন সংস্করণে)।


৬. MS Excel কি Online ব্যবহার করা যায়?

হ্যাঁ। Microsoft 365-এর মাধ্যমে Web Browser-এ Online ব্যবহার করা যায়।
