💻 Software কী?

Software হলো একগুচ্ছ Program বা নির্দেশনা, যা Hardware-কে কীভাবে কাজ করতে হবে তা নির্দেশ দেয়। Software দেখা যায় কিন্তু স্পর্শ করা যায় না।

🖥️ System Software

System Software কম্পিউটারের Hardware পরিচালনা করে এবং অন্যান্য Software চালানোর জন্য পরিবেশ তৈরি করে।

উদাহরণ:
  • ✅ Windows
  • ✅ Linux
  • ✅ macOS
  • ✅ Android

📱 Application Software

Application Software ব্যবহারকারীর নির্দিষ্ট কাজ সম্পন্ন করার জন্য তৈরি করা হয়।

উদাহরণ:
  • 📝 Microsoft Word
  • 📊 Microsoft Excel
  • 🌐 Google Chrome
  • 🎨 Paint
  • 🎵 VLC Media Player

📊 System Software ও Application Software-এর পার্থক্য

System Software Application Software
Hardware পরিচালনা করে ব্যবহারকারীর কাজ সম্পন্ন করে
কম্পিউটারের জন্য অপরিহার্য প্রয়োজন অনুযায়ী ব্যবহার করা হয়
Windows, Linux MS Word, Excel, Chrome

💡 পরীক্ষার জন্য গুরুত্বপূর্ণ তথ্য

  • ✅ Software দেখা যায়, কিন্তু স্পর্শ করা যায় না।
  • ✅ Windows একটি System Software।
  • ✅ MS Word একটি Application Software।
  • ✅ Chrome একটি Application Software।
  • ✅ Software ছাড়া Hardware কাজ করতে পারে না।

❓ Software MCQ

  1. Software কী?
    A) কম্পিউটারের দৃশ্যমান অংশ
    B) একগুচ্ছ Program
    C) Input Device
    D) Output Device
    উত্তর: B) একগুচ্ছ Program

  2. নিচের কোনটি System Software?
    A) MS Word
    B) Google Chrome
    C) Windows
    D) Paint
    উত্তর: C) Windows

  3. MS Word কোন ধরনের Software?
    A) System Software
    B) Application Software
    C) Utility Software
    D) Firmware
    উত্তর: B) Application Software

  4. নিচের কোনটি Application Software?
    A) Linux
    B) Windows
    C) Excel
    D) Android
    উত্তর: C) Excel

  5. Software সম্পর্কে সঠিক বক্তব্য কোনটি?
    A) স্পর্শ করা যায়
    B) দেখা ও স্পর্শ করা যায়
    C) দেখা যায় কিন্তু স্পর্শ করা যায় না
    D) কোনটিই নয়
    উত্তর: C) দেখা যায় কিন্তু স্পর্শ করা যায় না

  6. কম্পিউটার চালানোর জন্য কোন Software প্রয়োজন?
    A) Application Software
    B) System Software
    C) Game Software
    D) Browser
    উত্তর: B) System Software

  7. Google Chrome কী?
    A) Operating System
    B) Application Software
    C) Hardware
    D) CPU
    উত্তর: B) Application Software

  8. Android কী?
    A) Hardware
    B) Application Software
    C) System Software (Operating System)
    D) Browser
    উত্তর: C) System Software (Operating System)

  9. Hardware কাজ করার জন্য কী প্রয়োজন?
    A) Software
    B) Printer
    C) Mouse
    D) Scanner
    উত্তর: A) Software

  10. নিচের কোনটি Software নয়?
    A) Windows
    B) Keyboard
    C) MS Excel
    D) Chrome
    উত্তর: B) Keyboard
