💡 পরীক্ষার জন্য গুরুত্বপূর্ণ Shortcut Keys

  • ✅ Ctrl + C = Copy
  • ✅ Ctrl + V = Paste
  • ✅ Ctrl + X = Cut
  • ✅ Ctrl + Z = Undo
  • ✅ Ctrl + S = Save
  • ✅ Ctrl + P = Print
  • ✅ Ctrl + B = Bold
  • ✅ F5 = Slide Show
  • ✅ Ctrl + T = New Tab / Excel Table (প্রসঙ্গভেদে)

❓ Computer Shortcut Keys MCQ

  1. Copy করার Shortcut Key কোনটি?
    A) Ctrl + X
    B) Ctrl + C
    C) Ctrl + V
    D) Ctrl + Z
    উত্তর: B) Ctrl + C

  2. Paste করার Shortcut Key কোনটি?
    A) Ctrl + P
    B) Ctrl + V
    C) Ctrl + X
    D) Ctrl + B
    উত্তর: B) Ctrl + V

  3. Cut করার Shortcut Key কোনটি?
    A) Ctrl + X
    B) Ctrl + C
    C) Ctrl + V
    D) Ctrl + A
    উত্তর: A) Ctrl + X

  4. Undo করার Shortcut Key কোনটি?
    A) Ctrl + U
    B) Ctrl + Z
    C) Ctrl + Y
    D) Ctrl + S
    উত্তর: B) Ctrl + Z

  5. Redo করার Shortcut Key কোনটি?
    A) Ctrl + Z
    B) Ctrl + Y
    C) Ctrl + P
    D) Ctrl + F
    উত্তর: B) Ctrl + Y

  6. নতুন Document তৈরির Shortcut Key কী?
    A) Ctrl + N
    B) Ctrl + O
    C) Ctrl + S
    D) Ctrl + T
    উত্তর: A) Ctrl + N

  7. Document Save করার Shortcut Key কী?
    A) Ctrl + P
    B) Ctrl + S
    C) Ctrl + O
    D) Ctrl + H
    উত্তর: B) Ctrl + S

  8. Slide Show শুরু করার Shortcut Key কী?
    A) Ctrl + M
    B) F5
    C) F2
    D) Ctrl + L
    উত্তর: B) F5

  9. নতুন Browser Tab খোলার Shortcut Key কী?
    A) Ctrl + W
    B) Ctrl + T
    C) Ctrl + H
    D) Ctrl + R
    উত্তর: B) Ctrl + T

  10. বর্তমান Window বন্ধ করার Shortcut Key কী?
    A) Alt + F4
    B) Alt + Tab
    C) Ctrl + W
    D) Win + E
    উত্তর: A) Alt + F4

  11. File Explorer খোলার Shortcut Key কী?
    A) Win + R
    B) Win + E
    C) Win + L
    D) Win + D
    উত্তর: B) Win + E

  12. Computer Lock করার Shortcut Key কী?
    A) Win + L
    B) Win + E
    C) Ctrl + L
    D) Alt + L
    উত্তর: A) Win + L
