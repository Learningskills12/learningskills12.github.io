🌐 Internet কী?

Internet হলো পৃথিবীর বিভিন্ন কম্পিউটার ও ডিভাইসকে সংযুক্ত করা একটি বিশাল বৈশ্বিক নেটওয়ার্ক। এর মাধ্যমে তথ্য আদান-প্রদান, যোগাযোগ, শিক্ষা, ব্যবসা ও বিনোদন করা যায়।

🌍 WWW (World Wide Web)

WWW (World Wide Web) হলো Internet-এর একটি সেবা, যার মাধ্যমে Web Browser ব্যবহার করে বিভিন্ন Website দেখা যায়।

পূর্ণরূপ: World Wide Web

🌎 Website ও Webpage

Website: একাধিক Webpage-এর সমষ্টিকে Website বলা হয়।

Webpage: Website-এর প্রতিটি পৃথক পৃষ্ঠাকে Webpage বলা হয়।

উদাহরণ:
Website → Learning Skills
Webpage → Home, Blog, Contact, Quiz

🌐 Web Browser

Web Browser হলো এমন একটি Software যার মাধ্যমে Internet-এর Website দেখা যায়।

উদাহরণ:
  • 🌍 Google Chrome
  • 🦊 Mozilla Firefox
  • 🧭 Microsoft Edge
  • 🧭 Safari
  • 🎭 Opera

🔎 Search Engine

Search Engine হলো এমন একটি Website বা সেবা যার মাধ্যমে Internet-এ তথ্য খোঁজা যায়।

উদাহরণ:
  • 🔍 Google
  • 🟦 Bing
  • 🦆 DuckDuckGo
  • 🌐 Yahoo

📧 E-mail (Electronic Mail)

E-mail হলো Internet-এর মাধ্যমে বার্তা, ছবি, ডকুমেন্ট ও অন্যান্য তথ্য পাঠানোর একটি মাধ্যম।

পূর্ণরূপ: Electronic Mail

  • ✅ Internet হলো Network-এর Network।
  • ✅ WWW-এর পূর্ণরূপ World Wide Web।
  • ✅ Google Chrome একটি Web Browser।
  • ✅ Google একটি Search Engine।
  • ✅ E-mail-এর পূর্ণরূপ Electronic Mail।

❓ Internet MCQ

  1. Internet কী?
    A) একটি Software
    B) একটি Operating System
    C) বিশ্বব্যাপী কম্পিউটার নেটওয়ার্ক
    D) একটি Browser
    উত্তর: C) বিশ্বব্যাপী কম্পিউটার নেটওয়ার্ক

  2. WWW-এর পূর্ণরূপ কী?
    A) World Web Wide
    B) World Wide Web
    C) Web World Wide
    D) Wide World Web
    উত্তর: B) World Wide Web

  3. Google Chrome কী?
    A) Search Engine
    B) Web Browser
    C) Operating System
    D) Website
    উত্তর: B) Web Browser

  4. Google কী?
    A) Browser
    B) Search Engine
    C) Operating System
    D) Hardware
    উত্তর: B) Search Engine

  5. E-mail-এর পূর্ণরূপ কী?
    A) Easy Mail
    B) Electronic Mail
    C) Electric Mail
    D) Express Mail
    উত্তর: B) Electronic Mail

  6. Website কী নিয়ে গঠিত?
    A) শুধুমাত্র ছবি
    B) Webpage
    C) Browser
    D) Hardware
    উত্তর: B) Webpage

  7. নিচের কোনটি Web Browser?
    A) Windows
    B) Google Chrome
    C) CPU
    D) RAM
    উত্তর: B) Google Chrome

  8. নিচের কোনটি Search Engine?
    A) Mozilla Firefox
    B) Microsoft Edge
    C) Google
    D) Opera
    উত্তর: C) Google

  9. Internet-এর মাধ্যমে কী করা যায়?
    A) তথ্য আদান-প্রদান
    B) E-mail পাঠানো
    C) Website দেখা
    D) উপরের সবগুলো
    উত্তর: D) উপরের সবগুলো

  10. Internet-কে কী বলা হয়?
    A) Computer
    B) Network of Networks
    C) Operating System
    D) Browser
    উত্তর: B) Network of Networks
