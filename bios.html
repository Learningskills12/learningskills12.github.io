🛠️ BIOS কী?

Basic Input Output System সম্পর্কে সম্পূর্ণ বাংলা নোট

🏠 Computer Home

🛠️ BIOS কী?

BIOS-এর পূর্ণরূপ Basic Input Output System। এটি Motherboard-এর একটি Firmware, যা কম্পিউটার চালু হওয়ার সঙ্গে সঙ্গে প্রথম কাজ শুরু করে। BIOS কম্পিউটারের Hardware পরীক্ষা করে এবং তারপর Operating System (যেমন Windows বা Linux) চালু করতে সাহায্য করে।

🎯 BIOS-এর প্রধান কাজ

  • 💻 কম্পিউটার চালু হওয়ার সাথে সাথে Hardware পরীক্ষা করা (POST)।
  • 🧠 CPU, RAM, Keyboard, Storage ইত্যাদি শনাক্ত করা।
  • 💿 Boot Device নির্বাচন করা।
  • ⚙️ Operating System চালু করার জন্য নিয়ন্ত্রণ হস্তান্তর করা।
  • 🔧 Hardware-এর মৌলিক সেটিংস সংরক্ষণ করা।

📜 BIOS-এর ইতিহাস

BIOS-এর ধারণা ১৯৭০-এর দশকের শেষ দিকে তৈরি হয় এবং IBM ১৯৮১ সালে তাদের Personal Computer (PC)-এ BIOS ব্যবহার করে। এরপর BIOS কম্পিউটারের একটি অপরিহার্য অংশ হয়ে ওঠে। বর্তমানে অনেক নতুন কম্পিউটারে BIOS-এর পরিবর্তে আরও আধুনিক UEFI (Unified Extensible Firmware Interface) ব্যবহৃত হলেও সাধারণভাবে একে এখনও BIOS বলা হয়।

⚙️ BIOS কীভাবে কাজ করে?

কম্পিউটারের Power Button চাপার সঙ্গে সঙ্গে BIOS চালু হয়। প্রথমে এটি POST (Power-On Self Test) সম্পন্ন করে, অর্থাৎ CPU, RAM, Keyboard, Storage Drive ও অন্যান্য Hardware ঠিকভাবে কাজ করছে কি না তা পরীক্ষা করে। এরপর BIOS Boot Device (SSD, HDD বা USB) খুঁজে Operating System চালু করে।

🧩 BIOS-এর প্রধান অংশগুলো

  • ⚡ POST (Power-On Self Test)
  • 💾 CMOS Setup Utility
  • 🔋 CMOS Battery
  • 📀 Boot Loader
  • ⚙️ BIOS Firmware Chip
  • 🔄 Boot Sequence Manager

⚙️ BIOS-এ কী কী Settings থাকে?

  • 🕒 Date & Time সেট করা
  • 💽 Boot Priority পরিবর্তন
  • 🔒 BIOS Password সেট করা
  • 🖥️ CPU ও RAM সম্পর্কিত Settings
  • 🧊 Fan Control
  • 💾 SATA / NVMe Configuration
  • 🔌 USB Enable বা Disable করা

🔄 BIOS ও UEFI

UEFI হলো BIOS-এর আধুনিক সংস্করণ। এটি দ্রুত Boot হয়, বড় Storage Drive সমর্থন করে, সুন্দর Graphical Interface প্রদান করে এবং Secure Boot-এর মতো উন্নত Security Feature সমর্থন করে। বর্তমানে অধিকাংশ নতুন Desktop ও Laptop-এ UEFI ব্যবহৃত হয়।

📂 BIOS-এর প্রকারভেদ

  • Legacy BIOS — পুরোনো ধরনের BIOS, যা MBR Partition System ব্যবহার করে।
  • UEFI BIOS — আধুনিক Firmware, যা GPT Partition, Secure Boot এবং Fast Boot সমর্থন করে।

⭐ BIOS-এর বৈশিষ্ট্য

  • 🛠️ Motherboard-এর ROM/Flash Memory-তে সংরক্ষিত থাকে।
  • ⚡ কম্পিউটার চালু হওয়ার সাথে সাথেই কাজ শুরু করে।
  • 💻 Hardware ও Operating System-এর মধ্যে সংযোগ স্থাপন করে।
  • 🔋 CMOS-এর মাধ্যমে বিভিন্ন Settings সংরক্ষণ করে।
  • 🔄 প্রয়োজনে BIOS Update (Flash) করা যায়।
  • 🔐 Password Protection সমর্থন করে।

✅ BIOS-এর সুবিধা

  • কম্পিউটারের Hardware সঠিকভাবে কাজ করছে কিনা পরীক্ষা করে।
  • Operating System Boot করতে সাহায্য করে।
  • Hardware Configuration সহজে পরিবর্তন করা যায়।
  • Boot Priority নির্ধারণ করা যায়।
  • System-এর স্থিতিশীলতা বজায় রাখতে সাহায্য করে।

❌ BIOS-এর অসুবিধা

  • Legacy BIOS-এর Boot Speed তুলনামূলক ধীর।
  • Graphical Interface থাকে না (পুরোনো BIOS-এ)।
  • ২TB-এর বেশি Storage Drive সম্পূর্ণভাবে সমর্থন করে না।
  • ভুলভাবে BIOS Update করলে Motherboard ক্ষতিগ্রস্ত হতে পারে।
  • Legacy BIOS-এ Security Feature তুলনামূলক কম।

💻 BIOS কোথায় ব্যবহার হয়?

  • 🖥️ Desktop Computer
  • 💼 Laptop
  • 🖧 Server
  • 🎮 Gaming Computer
  • 🏭 Industrial Computer
  • 💻 Workstation
  • 🧩 Embedded Computer System

🔄 BIOS Update (BIOS Flash) কী?

BIOS Update বা BIOS Flash হলো Motherboard-এর BIOS Firmware-এর নতুন সংস্করণ ইনস্টল করার প্রক্রিয়া। এর মাধ্যমে নতুন Hardware সমর্থন, Bug Fix, Security উন্নতি এবং System Stability বৃদ্ধি পায়। তবে BIOS Update করার সময় বিদ্যুৎ চলে গেলে বা ভুল File ব্যবহার করলে Motherboard ক্ষতিগ্রস্ত হতে পারে।

⚠️ BIOS সমস্যার লক্ষণ

  • 🖥️ কম্পিউটার চালু না হওয়া।
  • ⚫ Screen-এ কোনো Display না আসা।
  • 🔊 বারবার Beep Sound শোনা।
  • 💽 Operating System Boot না হওয়া।
  • 🔄 বারবার Restart হওয়া।
  • ⚙️ BIOS Settings বারবার Reset হয়ে যাওয়া।

💡 BIOS ব্যবহারের গুরুত্বপূর্ণ টিপস

  • ✅ প্রয়োজন ছাড়া BIOS Settings পরিবর্তন করো না।
  • ✅ শুধুমাত্র Motherboard প্রস্তুতকারকের Official BIOS Update ব্যবহার করো।
  • ✅ BIOS Update করার সময় বিদ্যুৎ সংযোগ স্থির রাখো।
  • ✅ BIOS Password ভুলে গেলে Reset করার নিয়ম জেনে নাও।
  • ✅ CMOS Battery দুর্বল হলে সময় ও Settings Reset হতে পারে।

🤔 জানেন কি?

  • 💡 BIOS-এর পূর্ণরূপ হলো Basic Input Output System।
  • 💡 BIOS Operating System-এর আগেই চালু হয়।
  • 💡 CMOS Battery খুলে অনেক সময় BIOS Settings Reset করা যায়।
  • 💡 আধুনিক কম্পিউটারে অধিকাংশ ক্ষেত্রে UEFI ব্যবহার করা হয়।
  • 💡 BIOS থেকেই Boot Order পরিবর্তন করা যায়।

🔐 BIOS Password কী?

BIOS Password হলো একটি নিরাপত্তা ব্যবস্থা, যার মাধ্যমে BIOS Settings-এ অননুমোদিত প্রবেশ রোধ করা যায়। অনেক প্রতিষ্ঠান ও অফিসে কম্পিউটারের নিরাপত্তা বাড়ানোর জন্য BIOS Password ব্যবহার করা হয়।

❓ BIOS সম্পর্কে FAQ

১. BIOS-এর পূর্ণরূপ কী?
উত্তর: Basic Input Output System।

২. BIOS কোথায় থাকে?
উত্তর: Motherboard-এর ROM বা Flash Memory Chip-এ।

৩. BIOS-এর প্রধান কাজ কী?
উত্তর: কম্পিউটারের Hardware পরীক্ষা করে Operating System চালু করা।

৪. BIOS এবং UEFI কি একই?
উত্তর: UEFI হলো BIOS-এর আধুনিক ও উন্নত সংস্করণ।

৫. BIOS Update কেন করা হয়?
উত্তর: নতুন Hardware সমর্থন, Bug Fix এবং Security উন্নত করার জন্য।
