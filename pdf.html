📄 PDF কী?

PDF (Portable Document Format) হলো একটি জনপ্রিয় Document File Format, যা Adobe Systems তৈরি করেছে। PDF-এর বিশেষ সুবিধা হলো—যে ডিভাইসেই খোলা হোক না কেন, ডকুমেন্টের Layout, Font, Image এবং Formatting একই থাকে।

🎯 PDF-এর প্রধান কাজ

  • 📄 Document নিরাপদভাবে সংরক্ষণ করা।
  • 📤 সহজে File Share করা।
  • 🖨️ একই Layout-এ Print করা।
  • 🔒 Password দিয়ে File Protect করা।
  • 📝 Form Fill ও Digital Signature ব্যবহার করা।
  • 📚 E-book ও Official Document প্রকাশ করা।

📜 PDF-এর ইতিহাস

PDF ১৯৯৩ সালে Adobe Systems দ্বারা তৈরি করা হয়। এর মূল উদ্দেশ্য ছিল এমন একটি File Format তৈরি করা যা যেকোনো Operating System ও Device-এ একইভাবে দেখা যায়। বর্তমানে PDF বিশ্বব্যাপী সবচেয়ে জনপ্রিয় Document Format-গুলোর একটি।

⚙️ PDF কীভাবে তৈরি করবেন?

  1. 📄 MS Word বা অন্য Document খুলুন।

  5. ✅ আপনার PDF File তৈরি হয়ে যাবে।

📜 PDF-এর বিস্তারিত ইতিহাস

PDF (Portable Document Format) ১৯৯৩ সালে Adobe Systems তৈরি করে। এর উদ্দেশ্য ছিল এমন একটি Document Format তৈরি করা, যা যেকোনো কম্পিউটার, মোবাইল বা অপারেটিং সিস্টেমে একইভাবে দেখা যাবে। বর্তমানে PDF একটি আন্তর্জাতিক মান (ISO 32000) হিসেবে স্বীকৃত এবং শিক্ষা, ব্যবসা, সরকারি ও ব্যক্তিগত কাজে ব্যাপকভাবে ব্যবহৃত হয়।
