🎨 CSS কী?

CSS (Cascading Style Sheets) হলো একটি Style Sheet Language, যা HTML দিয়ে তৈরি Web Page-এর রং, ডিজাইন, Layout, Font, Animation এবং Responsive Design নিয়ন্ত্রণ করতে ব্যবহৃত হয়।

🎯 CSS-এর প্রধান কাজ

  • 🎨 Web Page-এর ডিজাইন করা।
  • 🌈 রং ও Background পরিবর্তন করা।
  • 🔠 Font ও Text Style নিয়ন্ত্রণ করা।
  • 📦 Layout তৈরি করা।
  • 📱 Responsive Design তৈরি করা।
  • ✨ Animation ও Transition যোগ করা।

📜 CSS-এর সংক্ষিপ্ত ইতিহাস

CSS ১৯৯৬ সালে Håkon Wium Lie প্রস্তাব করেন। বর্তমানে CSS3 সবচেয়ে জনপ্রিয় সংস্করণ এবং আধুনিক Web Design-এর একটি গুরুত্বপূর্ণ প্রযুক্তি।

📜 CSS-এর বিস্তারিত ইতিহাস

CSS (Cascading Style Sheets) ১৯৯৬ সালে Håkon Wium Lie প্রস্তাব করেন। পরবর্তীতে W3C (World Wide Web Consortium) এটি মান (Standard) হিসেবে গ্রহণ করে। HTML ও CSS আলাদা করার ফলে Website-এর ডিজাইন পরিচালনা সহজ হয়ে যায়। বর্তমানে CSS3 আধুনিক Web Design-এর প্রধান প্রযুক্তি এবং এতে Flexbox, Grid, Animation, Transition, Transform, Media Query-এর মতো শক্তিশালী Feature রয়েছে।

২. CSS কে তৈরি করেন?

৩. CSS কি Programming Language?

৪. CSS-এর বর্তমান জনপ্রিয় সংস্করণ কী?

৫. CSS কোথায় ব্যবহার করা হয়?

Website ও Web Application-এর ডিজাইন ও Layout তৈরিতে ব্যবহার করা হয়।


৬. CSS কি HTML ছাড়া ব্যবহার করা যায়?

সাধারণভাবে না। CSS মূলত HTML বা অন্যান্য Markup Language-এর Element-কে Style করার জন্য ব্যবহৃত হয়।
