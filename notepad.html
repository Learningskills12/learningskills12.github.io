📝 Notepad কী?

Notepad হলো Microsoft Windows-এর সাথে থাকা একটি সহজ এবং হালকা (Lightweight) Text Editor। এর মাধ্যমে সাধারণ Text File তৈরি, সম্পাদনা (Edit) এবং সংরক্ষণ (Save) করা যায়। এটি মূলত .txt ফাইল তৈরির জন্য ব্যবহৃত হয় এবং Programming-এর সাধারণ Source Code লেখার কাজেও ব্যবহৃত হয়।

🎯 Notepad-এর প্রধান উদ্দেশ্য

  • 📝 সাধারণ Text লেখা।
  • 💾 Text File সংরক্ষণ করা।
  • 💻 HTML, CSS, JavaScript, Python ইত্যাদির Source Code লেখা।
  • 📄 ছোট নোট বা তথ্য সংরক্ষণ করা।
  • ⚡ দ্রুত Text Edit করা।
  • 📋 Plain Text তৈরি করা।

📜 Notepad-এর ইতিহাস

Notepad প্রথম Microsoft Windows-এর প্রাথমিক সংস্করণগুলোর সাথে প্রকাশিত হয়। এরপর Windows-এর প্রতিটি নতুন সংস্করণে এটি উন্নত করা হয়েছে। Windows 11-এ Notepad নতুন Design, Dark Mode, Auto Save এবং Tab Support-এর মতো আধুনিক সুবিধা পেয়েছে।

⭐ Notepad-এর বৈশিষ্ট্য

  • ⚡ খুব দ্রুত চালু হয়।
  • 💾 খুব কম RAM ব্যবহার করে।
  • 📝 Plain Text Editor।
  • 📂 TXT File তৈরি ও সম্পাদনা করা যায়।
  • 💻 Programming Code লেখা যায়।
  • 🔍 Find ও Replace সুবিধা রয়েছে।
  • 🌙 Dark Mode সমর্থন করে (Windows 11)।
  • 📑 একাধিক Tab ব্যবহার করা যায় (নতুন সংস্করণে)।
  • 🖨️ Print করার সুবিধা রয়েছে।
  • 🌍 প্রায় সব Windows সংস্করণে পাওয়া যায়।

🖥️ Notepad-এর Interface

Notepad-এর Interface খুবই সহজ এবং ব্যবহারবান্ধব। নতুন ব্যবহারকারীরাও সহজে এটি ব্যবহার করতে পারেন।

  • 📌 Title Bar
  • 📋 Menu Bar
  • 📝 Text Editing Area
  • 📊 Status Bar
  • 🔍 Scroll Bar

📌 Notepad কোথায় ব্যবহার হয়?

  • 📝 সাধারণ নোট লেখার জন্য।
  • 💻 HTML, CSS ও JavaScript ফাইল তৈরির জন্য।
  • 🐍 Python Program লেখার জন্য।
  • ⚙️ Configuration File সম্পাদনার জন্য।
  • 📄 Log File পড়ার জন্য।
  • 📚 শিক্ষামূলক কাজে।
  • 🖥️ System Script লেখার জন্য।
  • 🔧 Software Development-এর প্রাথমিক পর্যায়ে।

File Menu-এর মাধ্যমে নতুন File তৈরি, পুরোনো File খোলা, Save, Print এবং Exit করা যায়।

🔤 Font Settings

Notepad-এ বিভিন্ন ধরনের Font, Font Style এবং Font Size নির্বাচন করা যায়, ফলে লেখা আরও পরিষ্কার ও পড়তে সুবিধাজনক হয়।

🔎 Zoom Feature

Zoom-এর মাধ্যমে লেখা বড় বা ছোট করে দেখা যায়। এটি চোখের আরাম এবং সহজে পড়ার জন্য উপকারী।

Windows 11-এর নতুন Notepad-এ Tabs Feature যোগ হয়েছে। এর মাধ্যমে একই উইন্ডোতে একাধিক Text File একসাথে খোলা ও পরিচালনা করা যায়।

  • ⚡ দ্রুত File পরিবর্তন

🔍 Find

Find অপশনের মাধ্যমে বড় Document-এর মধ্যে নির্দিষ্ট শব্দ দ্রুত খুঁজে পাওয়া যায়।

🔄 Replace

  • 🔄 Ctrl + H Shortcut।

📍 Go To

🕒 Time / Date
