💻 Computer 🪟 Windows 11

📝 Notepad কী?

Notepad হলো Microsoft Windows-এর একটি Plain Text Editor। এটি ব্যবহার করে সাধারণ Text File তৈরি, সম্পাদনা (Edit), সংরক্ষণ (Save) এবং Print করা যায়। Notepad মূলত .txt ফাইলের জন্য ব্যবহৃত হলেও HTML, CSS, JavaScript, Python, C, C++ সহ বিভিন্ন Programming Language-এর Source Code লেখার জন্যও এটি ব্যাপকভাবে ব্যবহৃত হয়।

🎯 Notepad-এর প্রধান উদ্দেশ্য

  • 📝 সাধারণ লেখা তৈরি করা।
  • 💾 Plain Text File সংরক্ষণ করা।
  • 💻 HTML, CSS, JavaScript-এর কোড লেখা।
  • 🐍 Python ও অন্যান্য Programming Code লেখা।
  • 📄 Notes ও To-Do List তৈরি করা।
  • ⚡ দ্রুত Text Edit করা।
  • 📂 Configuration File সম্পাদনা করা।

📜 Notepad-এর ইতিহাস

Notepad প্রথম Microsoft Windows-এর সাথে প্রকাশিত একটি সাধারণ Text Editor। Windows-এর প্রতিটি নতুন সংস্করণের সাথে এটি আরও উন্নত হয়েছে। বর্তমানে Windows 11-এ Notepad-এ Tabs, Dark Mode, Auto Save, উন্নত Find & Replace এবং Unicode Support-এর মতো আধুনিক সুবিধা যুক্ত হয়েছে।

⭐ Notepad-এর বৈশিষ্ট্য (Features)

  • ⚡ খুব দ্রুত চালু হয়।
  • 💾 খুব কম RAM ও Storage ব্যবহার করে।
  • 📝 Plain Text Editor।
  • 📂 TXT File তৈরি ও সম্পাদনা করা যায়।
  • 💻 HTML, CSS, JavaScript, Python, C, C++ ইত্যাদির Source Code লেখা যায়।
  • 🔍 Find ও Replace সুবিধা রয়েছে।
  • 🌙 Dark Mode (Windows 11)।
  • 📑 Multiple Tabs Support।
  • 🖨️ Print করা যায়।
  • 🌍 Unicode সমর্থন করে।

Notepad-এর Interface খুবই সহজ এবং ব্যবহারবান্ধব। প্রধান অংশগুলো হলো—

Status Bar Line ও Column Number দেখায়।
Scroll Bar উপরে-নিচে ও ডানে-বামে যাওয়া যায়।

📂 File Menu

File Menu-এর মাধ্যমে File তৈরি, খোলা, সংরক্ষণ এবং প্রিন্ট করা যায়।

Print Document প্রিন্ট করে।
Exit Notepad বন্ধ করে।

📄 New File

New অপশন ব্যবহার করে একটি নতুন খালি Text File তৈরি করা যায়।

  • 📄 নতুন Document তৈরি করে।
  • ⌨️ Shortcut: Ctrl + N
  • ⚡ দ্রুত নতুন কাজ শুরু করা যায়।

📂 Open

Open অপশন ব্যবহার করে আগে থেকে Save করা File খোলা যায়।

  • 📁 TXT File খুলতে ব্যবহৃত হয়।
  • 💻 HTML, CSS, JS, XML, BAT ইত্যাদিও খোলা যায়।
  • ⌨️ Shortcut: Ctrl + O

💾 Save

Save অপশন বর্তমান File সংরক্ষণ করে।

  • ⌨️ Shortcut: Ctrl + S
  • 💾 দ্রুত পরিবর্তন সংরক্ষণ করা যায়।
  • 📄 একই File আপডেট হয়।

📁 Save As

Save As ব্যবহার করে অন্য নামে বা অন্য Location-এ File সংরক্ষণ করা যায়।

  • 📝 নতুন নাম দেওয়া যায়।
  • 📂 অন্য Folder-এ Save করা যায়।
  • 📄 File-এর Copy তৈরি হয়।

🖨️ Print

Print অপশন ব্যবহার করে লেখা Printer-এর মাধ্যমে কাগজে Print করা যায়।

  • ⌨️ Shortcut: Ctrl + P
  • 🖨️ Printer নির্বাচন করা যায়।
  • 📄 Page Setup অনুযায়ী Print হয়।

🔍 Find

বড় Document-এর মধ্যে নির্দিষ্ট শব্দ দ্রুত খুঁজে বের করতে Find ব্যবহার করা হয়।

  • ⌨️ Shortcut: Ctrl + F
  • ⚡ দ্রুত Search করা যায়।
  • 📖 বড় File-এর জন্য খুব উপকারী।

🔄 Replace

Replace অপশন ব্যবহার করে একটি শব্দের পরিবর্তে অন্য শব্দ একসাথে পরিবর্তন করা যায়।

  • ⌨️ Shortcut: Ctrl + H
  • ⚡ একসাথে বহু শব্দ পরিবর্তন করা যায়।
  • 📝 Editing দ্রুত হয়।

📍 Go To

Go To ব্যবহার করে নির্দিষ্ট Line Number-এ দ্রুত যাওয়া যায়।

  • ⌨️ Shortcut: Ctrl + G
  • 📄 বড় File-এ কাজ করতে সুবিধা হয়।

🕒 Time / Date

বর্তমান Date এবং Time Cursor-এর অবস্থানে Insert করা যায়।

  • ⌨️ Shortcut: F5
  • 📅 বর্তমান Date যোগ করে।
  • ⏰ বর্তমান Time যোগ করে।

↩ Undo ও ↪ Redo

ভুল করলে Undo দিয়ে আগের অবস্থায় ফেরা যায় এবং Redo দিয়ে সেই কাজ আবার ফিরিয়ে আনা যায়।

Shortcut কাজ
Ctrl + Z Undo
Ctrl + Y Redo

🗑 Delete ও ✅ Select All

  • 🗑 Delete Key দিয়ে নির্বাচিত লেখা মুছে ফেলা যায়।
  • ✅ Ctrl + A চাপলে পুরো Document নির্বাচন করা যায়।
  • ⚡ দ্রুত Editing করা যায়।

📃 Word Wrap

Word Wrap চালু থাকলে লেখা স্বয়ংক্রিয়ভাবে পরবর্তী লাইনে চলে যায়। বন্ধ থাকলে Horizontal Scroll ব্যবহার করতে হয়।

🔤 Font Settings

Font Option ব্যবহার করে Font Family, Font Style এবং Font Size পরিবর্তন করা যায়।

📊 Status Bar

Status Bar-এ বর্তমান Line Number, Column Number এবং Zoom Level দেখা যায়।

🔎 Zoom Feature

  • ➕ Zoom In
  • ➖ Zoom Out
  • 🔄 Restore Default Zoom

📑 Tabs Feature (Windows 11)

Windows 11-এর নতুন Notepad-এ Tabs Feature যোগ হয়েছে। এর মাধ্যমে একই Window-তে একাধিক File একসাথে খোলা ও পরিচালনা করা যায়।
