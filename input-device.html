⌨️ Input Devices

Learning Skills

⌨️ Keyboard

Keyboard হলো সবচেয়ে বেশি ব্যবহৃত Input Device। এর মাধ্যমে অক্ষর, সংখ্যা ও বিভিন্ন নির্দেশ কম্পিউটারে প্রবেশ করানো হয়।

🖱️ Mouse

Mouse একটি Pointing Device। এটি দিয়ে Click, Double Click, Drag & Drop এবং Scroll করা যায়।

📠 Scanner

Scanner কাগজের ছবি বা লেখা স্ক্যান করে ডিজিটাল ফাইলে রূপান্তর করে।

🎤 Microphone

Microphone-এর মাধ্যমে শব্দ বা Voice কম্পিউটারে Input করা হয়।

📷 Webcam

Webcam ব্যবহার করে Video Call, Online Meeting এবং Live Streaming করা হয়।

💡 Light Pen

Light Pen দিয়ে Monitor-এর উপর সরাসরি নির্দেশ বা Drawing করা যায়।

🕹️ Joystick

Joystick সাধারণত Computer Game এবং Flight Simulator নিয়ন্ত্রণ করতে ব্যবহৃত হয়।

📱 Touch Screen

Touch Screen-এ আঙুল দিয়ে স্পর্শ করেই কম্পিউটার বা মোবাইল নিয়ন্ত্রণ করা যায়।

❓ Input Device MCQ

  1. Keyboard কী ধরনের Device?
    উত্তর: Input Device

  2. Mouse কী কাজে ব্যবহৃত হয়?
    উত্তর: Pointing ও Selection

  3. কোন Device কাগজকে Digital File-এ রূপান্তর করে?
    উত্তর: Scanner

  4. শব্দ ইনপুট করতে কোন Device ব্যবহৃত হয়?
    উত্তর: Microphone

  5. Video Call-এর জন্য কোন Device ব্যবহার করা হয়?
    উত্তর: Webcam

  6. Game খেলার জন্য কোন Input Device বেশি ব্যবহৃত হয়?
    উত্তর: Joystick

  7. Touch Screen কী?
    উত্তর: একটি Input Device

  8. Light Pen কোথায় ব্যবহার করা হয়?
    উত্তর: Screen-এ Drawing ও Selection

  9. Typing করার জন্য কোন Device ব্যবহৃত হয়?
    উত্তর: Keyboard

  10. সবচেয়ে জনপ্রিয় Pointing Device কোনটি?
    উত্তর: Mouse
