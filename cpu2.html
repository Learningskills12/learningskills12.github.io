🧠 CPU (Central Processing Unit)

Learning Skills

🧠 CPU কী?

CPU (Central Processing Unit) হলো কম্পিউটারের প্রধান প্রসেসিং অংশ। এটিকে কম্পিউটারের "Brain of the Computer" বলা হয়। CPU সমস্ত নির্দেশ (Instructions) গ্রহণ করে, তথ্য প্রক্রিয়াকরণ করে এবং ফলাফল প্রদান করে।

🧮 ALU (Arithmetic Logic Unit)

ALU গাণিতিক (Addition, Subtraction, Multiplication, Division) এবং যৌক্তিক (AND, OR, NOT, Comparison) কাজ সম্পন্ন করে।

🎮 CU (Control Unit)

Control Unit (CU) কম্পিউটারের সকল অংশের কাজ নিয়ন্ত্রণ ও সমন্বয় করে। এটি Input, Memory, ALU এবং Output-এর মধ্যে নির্দেশ আদান-প্রদান পরিচালনা করে।

⚙️ Registers

Registers হলো CPU-এর ভেতরে থাকা অতি ক্ষুদ্র ও সবচেয়ে দ্রুতগতির Memory। এটি সাময়িকভাবে তথ্য সংরক্ষণ করে যাতে CPU দ্রুত কাজ করতে পারে।

🔄 CPU কীভাবে কাজ করে?

  1. 📥 Input Device থেকে তথ্য গ্রহণ করে।
  2. 💾 Memory থেকে প্রয়োজনীয় তথ্য আনে।
  3. 🧠 ALU ও CU-এর সাহায্যে তথ্য Process করে।
  4. 📤 ফলাফল Output Device-এ পাঠায়।

📊 CPU-এর প্রধান অংশ

অংশ কাজ
ALU গাণিতিক ও যৌক্তিক কাজ
CU সমস্ত কাজ নিয়ন্ত্রণ
Registers দ্রুত তথ্য সংরক্ষণ

💡 পরীক্ষার জন্য গুরুত্বপূর্ণ তথ্য

  • ✅ CPU = Central Processing Unit
  • ✅ CPU-কে Computer-এর Brain বলা হয়।
  • ✅ ALU গাণিতিক ও Logical কাজ করে।
  • ✅ CU সমস্ত অংশ নিয়ন্ত্রণ করে।
  • ✅ Registers হলো CPU-এর সবচেয়ে দ্রুত Memory।

❓ CPU MCQ

  1. CPU-এর পূর্ণরূপ কী?
    A) Central Program Unit
    B) Central Processing Unit
    C) Computer Processing Unit
    D) Control Processing Unit
    উত্তর: B) Central Processing Unit

  2. কম্পিউটারের Brain বলা হয় কাকে?
    A) RAM
    B) Hard Disk
    C) CPU
    D) ROM
    উত্তর: C) CPU

  3. ALU-এর প্রধান কাজ কী?
    A) Printing
    B) Arithmetic ও Logical Operation
    C) Scanning
    D) Data Storage
    উত্তর: B) Arithmetic ও Logical Operation

  4. CU-এর পূর্ণরূপ কী?
    A) Central Unit
    B) Control Unit
    C) Computer Unit
    D) Core Unit
    উত্তর: B) Control Unit

  5. Registers কোথায় থাকে?
    A) Hard Disk
    B) RAM
    C) CPU-এর ভিতরে
    D) ROM
    উত্তর: C) CPU-এর ভিতরে

  6. CPU কোন কাজটি করে?
    A) Data Processing
    B) Food Processing
    C) Video Recording
    D) Printing Only
    উত্তর: A) Data Processing

  7. CPU-এর সবচেয়ে দ্রুত Memory কোনটি?
    A) Hard Disk
    B) Registers
    C) DVD
    D) Pen Drive
    উত্তর: B) Registers

  8. Logical Operation কোথায় সম্পন্ন হয়?
    A) CU
    B) ALU
    C) RAM
    D) ROM
    উত্তর: B) ALU

  9. CPU-এর Control অংশ কোনটি?
    A) CU
    B) ALU
    C) ROM
    D) SSD
    উত্তর: A) CU

  10. CPU-এর প্রধান অংশ কয়টি?
    A) ২টি
    B) ৩টি
    C) ৪টি
    D) ৫টি
    উত্তর: B) ৩টি (ALU, CU, Registers)
