⚡ JavaScript কী?

JavaScript (JS) হলো একটি জনপ্রিয় Programming Language, যা Website-কে Interactive ও Dynamic করে। HTML Website-এর Structure তৈরি করে, CSS Design করে এবং JavaScript Website-এ বিভিন্ন কার্যক্রম (Functionality) যোগ করে।

🎯 JavaScript-এর প্রধান কাজ

  • 🖱️ Button Click-এর প্রতিক্রিয়া দেওয়া।
  • 📝 Form Validation করা।
  • 🔄 Dynamic Content পরিবর্তন করা।
  • 🎞️ Animation ও Effect যোগ করা।
  • 📡 Server থেকে Data আনা (API)।
  • 🎮 Interactive Web Application তৈরি করা।

📜 JavaScript-এর সংক্ষিপ্ত ইতিহাস

JavaScript ১৯৯৫ সালে Brendan Eich তৈরি করেন। এটি প্রথমে Netscape Navigator Browser-এর জন্য তৈরি হলেও বর্তমানে সব আধুনিক Browser এবং Server-side Development (Node.js)-এ ব্যবহৃত হয়।

📜 JavaScript-এর বিস্তারিত ইতিহাস

JavaScript ১৯৯৫ সালে Brendan Eich মাত্র ১০ দিনের মধ্যে তৈরি করেন। এটি প্রথমে Mocha নামে পরিচিত ছিল, পরে LiveScript এবং শেষ পর্যন্ত JavaScript নাম দেওয়া হয়। বর্তমানে এটি বিশ্বের অন্যতম জনপ্রিয় Programming Language এবং Front-end, Back-end, Mobile App, Desktop App ও Server Development-এ ব্যাপকভাবে ব্যবহৃত হয়।
