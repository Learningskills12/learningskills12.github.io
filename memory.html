💾 Memory কী?

Memory হলো কম্পিউটারের এমন একটি অংশ যেখানে তথ্য (Data), নির্দেশ (Instructions) এবং প্রোগ্রাম সংরক্ষণ করা হয়। CPU প্রয়োজন অনুযায়ী Memory থেকে তথ্য গ্রহণ করে কাজ সম্পন্ন করে।

⚡ RAM (Random Access Memory)

RAM হলো একটি Temporary Memory। কম্পিউটার চালু থাকা অবস্থায় এটি তথ্য সংরক্ষণ করে। বিদ্যুৎ চলে গেলে RAM-এর তথ্য মুছে যায়।

💿 ROM (Read Only Memory)

ROM হলো Permanent Memory। এতে সংরক্ষিত তথ্য বিদ্যুৎ চলে গেলেও নষ্ট হয় না। কম্পিউটার চালু হওয়ার জন্য প্রয়োজনীয় নির্দেশ ROM-এ সংরক্ষিত থাকে।

🚀 Cache Memory

Cache Memory হলো খুব দ্রুতগতির Memory। এটি CPU-এর কাছাকাছি থাকে এবং বারবার ব্যবহৃত তথ্য সংরক্ষণ করে কম্পিউটারের গতি বৃদ্ধি করে।

📦 Primary Memory

Primary Memory হলো CPU-এর সরাসরি ব্যবহৃত Memory। এর প্রধান দুটি অংশ হলো RAM এবং ROM।

💽 Secondary Memory

Secondary Memory-তে তথ্য দীর্ঘ সময়ের জন্য সংরক্ষণ করা হয়। যেমন: Hard Disk, SSD, Pen Drive, CD/DVD এবং Memory Card।

📊 RAM ও ROM-এর তুলনা

RAM ROM
অস্থায়ী (Temporary) স্থায়ী (Permanent)
বিদ্যুৎ চলে গেলে তথ্য মুছে যায় তথ্য মুছে যায় না
Read ও Write করা যায় মূলত Read করা হয়

💡 পরীক্ষার জন্য গুরুত্বপূর্ণ তথ্য

  • ✅ RAM = Temporary Memory
  • ✅ ROM = Permanent Memory
  • ✅ Cache Memory সবচেয়ে দ্রুত।
  • ✅ Hard Disk হলো Secondary Memory।
  • ✅ RAM ও ROM হলো Primary Memory।

❓ Memory MCQ

  1. RAM কী ধরনের Memory?
    A) Permanent
    B) Temporary
    C) Secondary
    D) Optical
    উত্তর: B) Temporary

  2. ROM-এর পূর্ণরূপ কী?
    A) Random Output Memory
    B) Read Only Memory
    C) Read Output Memory
    D) Random Only Memory
    উত্তর: B) Read Only Memory

  3. সবচেয়ে দ্রুত Memory কোনটি?
    A) Hard Disk
    B) RAM
    C) Cache Memory
    D) DVD
    উত্তর: C) Cache Memory

  4. Hard Disk কোন ধরনের Memory?
    A) Primary
    B) Secondary
    C) Cache
    D) RAM
    উত্তর: B) Secondary

  5. বিদ্যুৎ চলে গেলে কোন Memory-এর তথ্য মুছে যায়?
    A) ROM
    B) RAM
    C) SSD
    D) CD
    উত্তর: B) RAM

  6. Primary Memory-এর অন্তর্ভুক্ত কোনটি?
    A) RAM
    B) ROM
    C) উভয়ই
    D) কোনটিই নয়
    উত্তর: C) উভয়ই

  7. Pen Drive কোন ধরনের Memory?
    A) Primary
    B) Secondary
    C) Cache
    D) Register
    উত্তর: B) Secondary

  8. ROM-এর তথ্য কখন নষ্ট হয়?
    A) বিদ্যুৎ গেলে
    B) কম্পিউটার বন্ধ হলে
    C) সাধারণভাবে নষ্ট হয় না
    D) Restart করলে
    উত্তর: C) সাধারণভাবে নষ্ট হয় না

  9. CPU-এর সবচেয়ে কাছাকাছি Memory কোনটি?
    A) DVD
    B) Cache Memory
    C) Hard Disk
    D) Pen Drive
    উত্তর: B) Cache Memory

  10. SSD কী?
    A) Input Device
    B) Output Device
    C) Secondary Storage Device
    D) Processing Device
    উত্তর: C) Secondary Storage Device
