📜 RAM-এর ইতিহাস

কম্পিউটারের শুরুতে RAM হিসেবে Magnetic Core Memory ব্যবহার করা হতো। এটি আকারে বড় এবং ধীরগতির ছিল।

পরবর্তীতে DRAM (Dynamic RAM) এবং SRAM (Static RAM) আবিষ্কারের মাধ্যমে কম্পিউটার অনেক দ্রুত কাজ করতে শুরু করে।

বর্তমানে DDR3, DDR4 এবং DDR5 প্রযুক্তির RAM সবচেয়ে বেশি ব্যবহৃত হয়। প্রতিটি নতুন সংস্করণ আগের তুলনায় দ্রুত, শক্তি-সাশ্রয়ী এবং উন্নত।

⚙️ RAM কীভাবে কাজ করে?

আপনি যখন কোনো সফটওয়্যার (যেমন Google Chrome, Microsoft Word বা একটি গেম) চালু করেন, তখন সেই সফটওয়্যারের প্রয়োজনীয় ডেটা Hard Disk বা SSD থেকে RAM-এ আসে।

CPU এরপর RAM থেকে খুব দ্রুত সেই তথ্য পড়ে এবং প্রসেস করে। তাই RAM যত দ্রুত হবে, কম্পিউটারও তত দ্রুত কাজ করবে।

💡 সহজ উদাহরণ: Hard Disk হলো একটি **বইয়ের আলমারি** আর RAM হলো **পড়ার টেবিল**। আপনি বই আলমারি থেকে এনে টেবিলে রাখলে সহজে পড়তে পারেন। ঠিক তেমনি CPU আগে RAM-এ ডেটা নিয়ে কাজ করে।

📊 DDR3 vs DDR4 vs DDR5

বৈশিষ্ট্য DDR3 DDR4 DDR5

ব্যবহার পুরনো কম্পিউটার বর্তমান কম্পিউটার নতুন ও উচ্চক্ষমতার কম্পিউটার

📝 পরীক্ষার জন্য গুরুত্বপূর্ণ তথ্য

  • RAM-এর পূর্ণরূপ: Random Access Memory
  • RAM একটি Volatile Memory।
  • বিদ্যুৎ চলে গেলে RAM-এর ডেটা মুছে যায়।
  • RAM CPU-কে দ্রুত তথ্য সরবরাহ করে।
  • RAM যত বেশি হবে, Multitasking তত ভালো হবে।
  • বর্তমানে DDR4 ও DDR5 সবচেয়ে বেশি ব্যবহৃত RAM।

⚠️ সাধারণ ভুল ধারণা

অনেকে মনে করেন RAM-এ ছবি, ভিডিও বা গান স্থায়ীভাবে সংরক্ষণ করা যায়। এটি ভুল ধারণা। RAM শুধুমাত্র অস্থায়ীভাবে তথ্য সংরক্ষণ করে। স্থায়ীভাবে তথ্য সংরক্ষণের জন্য Hard Disk বা SSD ব্যবহার করা হয়।

🎯 উপসংহার

RAM যত বেশি হবে, কম্পিউটার তত দ্রুত ও স্মুথভাবে কাজ করবে। তবে RAM স্থায়ী মেমোরি নয়।
