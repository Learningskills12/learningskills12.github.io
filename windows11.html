💻 Operating System

🪟 Windows 11 কী?

Windows 11 হলো Microsoft-এর তৈরি একটি আধুনিক Operating System যা ৫ অক্টোবর ২০২১ সালে প্রকাশিত হয়। এটি Windows 10-এর উত্তরসূরি এবং নতুন ডিজাইন, উন্নত নিরাপত্তা, দ্রুত কর্মক্ষমতা ও আধুনিক User Interface নিয়ে এসেছে।

🎯 Windows 11-এর প্রধান উদ্দেশ্য

  • 💻 কম্পিউটার পরিচালনা করা।
  • 📂 File ও Folder ব্যবস্থাপনা।
  • 🌐 Internet ব্যবহার করা।
  • 🎮 Gaming-এর উন্নত অভিজ্ঞতা প্রদান।
  • 📚 শিক্ষা ও অফিসের কাজ সহজ করা।
  • 🔒 উন্নত নিরাপত্তা নিশ্চিত করা।

📜 Windows 11-এর ইতিহাস

Microsoft ২৪ জুন ২০২১ সালে Windows 11 ঘোষণা করে এবং ৫ অক্টোবর ২০২১ সালে এটি সাধারণ ব্যবহারকারীদের জন্য প্রকাশ করে। Windows 10-এর তুলনায় এতে নতুন Start Menu, কেন্দ্রে Taskbar, Widgets, Snap Layouts, উন্নত Security এবং আধুনিক Interface যোগ করা হয়েছে।

  • 🪟 নতুন ও আকর্ষণীয় User Interface
  • 🎨 Center Start Menu

Windows 11 চালু হলে প্রথম যে Screen দেখা যায় তাকে Desktop বলা হয়। এখান থেকে বিভিন্ন সফটওয়্যার, ফাইল ও ফোল্ডার সহজে ব্যবহার করা যায়।
