🌐 HTML কী?

HTML (HyperText Markup Language) হলো একটি Markup Language যা Web Page তৈরি করার জন্য ব্যবহৃত হয়। HTML ওয়েব পেজের কাঠামো (Structure) তৈরি করে এবং Browser-কে জানায় কোন তথ্য কীভাবে প্রদর্শন করতে হবে।

🎯 HTML-এর প্রধান কাজ

  • 📄 Web Page-এর Structure তৈরি করা।
  • 📝 Text প্রদর্শন করা।
  • 🖼️ Image যোগ করা।
  • 🔗 Link তৈরি করা।
  • 📋 Table ও List তৈরি করা।
  • 📝 Form তৈরি করা।
  • 🎥 Audio ও Video যুক্ত করা।

📜 HTML-এর সংক্ষিপ্ত ইতিহাস

HTML ১৯৯১ সালে Tim Berners-Lee তৈরি করেন। বর্তমানে HTML-এর সর্বশেষ প্রধান সংস্করণ হলো HTML5, যা আধুনিক Web Development-এর ভিত্তি।

⭐ HTML-এর বৈশিষ্ট্য

  • 🌐 Web Page তৈরির মূল ভাষা।
  • 📖 শেখা সহজ।
  • 💻 সব Browser সমর্থন করে।
  • 🎨 CSS ও JavaScript-এর সাথে কাজ করে।
  • 📱 Responsive Website তৈরিতে সহায়ক।
  • ⚡ দ্রুত লোড হয়।

🏷️ HTML-এর গুরুত্বপূর্ণ Tag

Tag কাজ
<html> সম্পূর্ণ HTML Document নির্দেশ করে
<head> Document-এর তথ্য সংরক্ষণ করে
<title> Browser Tab-এর শিরোনাম নির্ধারণ করে
<body> ওয়েব পেজের দৃশ্যমান অংশ
<h1> - <h6> Heading তৈরি করে
<p> Paragraph তৈরি করে
<a> Hyperlink তৈরি করে
<img> Image প্রদর্শন করে
<table> Table তৈরি করে
<form> Form তৈরি করে

📌 HTML সম্পর্কে গুরুত্বপূর্ণ তথ্য

  • 👨‍💻 Tim Berners-Lee HTML তৈরি করেন।

⚙️ HTML কীভাবে কাজ করে?

  1. 📝 Developer HTML Code লিখেন।
  2. 💾 Code .html ফাইলে সংরক্ষণ করা হয়।
  3. 🌐 Browser (Chrome, Firefox, Edge, Safari) ফাইলটি পড়ে।
  4. 🖥️ Browser HTML Tag বিশ্লেষণ (Parse) করে।
  5. 📄 Web Page ব্যবহারকারীর সামনে প্রদর্শিত হয়।

📜 HTML-এর বিস্তারিত ইতিহাস

HTML (HyperText Markup Language) ১৯৯১ সালে Tim Berners-Lee তৈরি করেন। এটি World Wide Web (WWW)-এর ভিত্তি হিসেবে ব্যবহৃত হয়। পরবর্তীতে HTML 2.0, HTML 3.2, HTML 4.01 এবং HTML5 প্রকাশিত হয়। বর্তমানে HTML5 সবচেয়ে বেশি ব্যবহৃত সংস্করণ এবং এটি আধুনিক Website ও Web Application তৈরির মূল ভিত্তি।

২. HTML কে তৈরি করেছেন?

৩. HTML কি Programming Language?

৪. HTML5 কী?

৫. HTML কি একাই Website তৈরি করতে পারে?

HTML দিয়ে কাঠামো তৈরি করা যায়, তবে সুন্দর ডিজাইন ও ইন্টারঅ্যাকটিভ ফিচারের জন্য CSS এবং JavaScript ব্যবহৃত হয়।
