💾 RAM কী?

RAM (Random Access Memory)

RAM হলো কম্পিউটারের অস্থায়ী (Temporary) মেমোরি। কম্পিউটার চালু থাকা অবস্থায় যেসব তথ্য ও প্রোগ্রাম ব্যবহার হয় সেগুলো RAM-এ সংরক্ষিত থাকে।

সহজ ভাষায়:
RAM যত বেশি হবে, কম্পিউটার তত দ্রুত একসাথে অনেক কাজ করতে পারবে।

RAM-এর কাজ

  • একসাথে একাধিক সফটওয়্যার চালাতে সাহায্য করে।
  • কম্পিউটারের গতি বাড়ায়।
  • গেম দ্রুত লোড করে।
  • ভিডিও এডিটিং সহজ করে।
  • ব্রাউজারে অনেক Tab খুললেও কম সমস্যা হয়।

RAM-এর প্রকারভেদ

ধরণ ব্যবহার
DDR3 পুরনো কম্পিউটার
DDR4 বর্তমানে সবচেয়ে বেশি ব্যবহৃত
DDR5 নতুন ও দ্রুত প্রযুক্তি

কত GB RAM ভালো?

  • ৪ GB → সাধারণ কাজ
  • ৮ GB → ছাত্রছাত্রী ও অফিস
  • ১৬ GB → Gaming ও Editing
  • ৩২ GB+ → Professional Work

মনে রাখুন

RAM কম্পিউটারের Speed বাড়াতে সাহায্য করে, কিন্তু এটি Permanent Storage নয়। কম্পিউটার বন্ধ করলে RAM-এর ডেটা মুছে যায়।

⭐ RAM-এর বৈশিষ্ট্য

  • Temporary Memory
  • খুব দ্রুত Data Read ও Write করতে পারে।
  • বিদ্যুৎ চলে গেলে Data মুছে যায়।
  • CPU-এর সাথে সরাসরি কাজ করে।

💻 RAM-এর সুবিধা

  • ✔️ কম্পিউটার দ্রুত কাজ করে
  • ✔️ একসাথে অনেক সফটওয়্যার চালানো যায়
  • ✔️ Gaming Performance উন্নত হয়
  • ✔️ Video Editing দ্রুত হয়
  • ✔️ System Hang কম হয়

❌ RAM-এর অসুবিধা

  • Permanent Storage নয়।
  • বিদ্যুৎ চলে গেলে Data মুছে যায়।
  • RAM কম হলে কম্পিউটার Slow হয়।

🎯 উপসংহার
