📄 MS Word কী?

MS Word (Microsoft Word) হলো Microsoft Corporation-এর তৈরি একটি জনপ্রিয় Word Processing Software। এটি ব্যবহার করে চিঠি, আবেদনপত্র, রিপোর্ট, জীবনবৃত্তান্ত (CV), বই, প্রশ্নপত্র, নোট, প্রজেক্ট এবং বিভিন্ন ধরনের ডকুমেন্ট তৈরি, সম্পাদনা, সংরক্ষণ ও প্রিন্ট করা যায়।

🎯 MS Word-এর প্রধান কাজ

  • 📝 Document তৈরি করা।
  • ✏️ লেখা সম্পাদনা (Editing) করা।
  • 🎨 Text Formatting করা।
  • 🖼️ ছবি, Table ও Shape যোগ করা।
  • 📄 Page Layout পরিবর্তন করা।
  • 🖨️ Document Print করা।
  • 💾 Document Save ও PDF হিসেবে সংরক্ষণ করা।

📜 MS Word-এর ইতিহাস

Microsoft প্রথম ১৯৮৩ সালে MS Word প্রকাশ করে। শুরুতে এটি MS-DOS-এর জন্য তৈরি হলেও পরে Windows এবং macOS-এর জন্য উন্নত সংস্করণ প্রকাশ করা হয়। বর্তমানে Microsoft 365-এর অংশ হিসেবে MS Word বিশ্বের অন্যতম জনপ্রিয় Word Processing Software।
