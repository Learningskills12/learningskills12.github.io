📝 MS Word কী?

Microsoft Word হলো Microsoft-এর তৈরি একটি জনপ্রিয় Word Processing Software। এটি দিয়ে চিঠি, আবেদনপত্র, রিপোর্ট, প্রশ্নপত্র, বই, নোটস এবং বিভিন্ন ধরনের ডকুমেন্ট তৈরি ও সম্পাদনা করা যায়।

🎯 MS Word-এর ব্যবহার

  • 📄 চিঠি লেখা
  • 📝 আবেদনপত্র তৈরি
  • 📚 নোটস ও বই লেখা
  • 📊 রিপোর্ট তৈরি
  • 🖨️ প্রিন্ট করার জন্য ডকুমেন্ট তৈরি
  • 📋 প্রশ্নপত্র তৈরি

🖥️ MS Word কীভাবে চালু করবেন?

Windows-এ:

Start Menu → Microsoft Office → Microsoft Word

অথবা

Windows Key চাপুন → Word লিখুন → Enter চাপুন।

📂 New Document

নতুন ডকুমেন্ট তৈরি করতে:

File → New → Blank Document

অথবা
Shortcut Key: Ctrl + N

💾 Save ও Save As

ডকুমেন্ট সংরক্ষণ করতে:

File → Save
Shortcut: Ctrl + S

অন্য নামে সংরক্ষণ করতে:
File → Save As

📂 Open Document

আগের ডকুমেন্ট খুলতে:

File → Open
Shortcut: Ctrl + O

💡 পরীক্ষার জন্য গুরুত্বপূর্ণ তথ্য

  • ✅ MS Word একটি Application Software।
  • ✅ এটি Word Processing Software।
  • ✅ Ctrl + N = New Document
  • ✅ Ctrl + S = Save
  • ✅ Ctrl + O = Open
  • ✅ Microsoft Word দিয়ে ডকুমেন্ট তৈরি ও সম্পাদনা করা যায়।

✍️ Text Formatting

  • Bold → Ctrl + B
  • Italic → Ctrl + I
  • Underline → Ctrl + U
  • Copy → Ctrl + C
  • Cut → Ctrl + X
  • Paste → Ctrl + V
  • Select All → Ctrl + A
  • Undo → Ctrl + Z
  • Redo → Ctrl + Y

📊 Table Insert

Table তৈরি করতে:

Insert → Table

প্রয়োজন অনুযায়ী Row এবং Column নির্বাচন করুন।

🖼️ Picture Insert

ছবি যোগ করতে:

Insert → Pictures → This Device

তারপর ছবি নির্বাচন করে Insert চাপুন।

🖨️ Print Document

Document Print করতে:

File → Print
Shortcut Key: Ctrl + P
