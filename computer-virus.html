🛡 Antivirus

🦠 Computer Virus কী?

Computer Virus হলো একটি ক্ষতিকারক (Malicious) Program যা কম্পিউটারের ফাইল, সফটওয়্যার বা Operating System-এর ক্ষতি করতে পারে। এটি নিজের কপি তৈরি করে অন্য ফাইল বা কম্পিউটারে ছড়িয়ে পড়তে সক্ষম।

🎯 Computer Virus-এর উদ্দেশ্য

  • 💻 কম্পিউটারের ক্ষতি করা।
  • 📂 গুরুত্বপূর্ণ ফাইল নষ্ট করা।
  • 🔑 ব্যক্তিগত তথ্য চুরি করা।
  • 🌐 নেটওয়ার্কে ছড়িয়ে পড়া।
  • 💰 অর্থ আদায়ের জন্য ফাইল লক করা।
  • ⚠️ সিস্টেমকে ধীর বা অচল করে দেওয়া।

📜 Computer Virus-এর ইতিহাস

১৯৭১ সালে তৈরি Creeper প্রোগ্রামকে প্রথম কম্পিউটার ভাইরাসের পূর্বসূরি হিসেবে ধরা হয়। পরে ১৯৮৬ সালে Brain Virus প্রথম PC Virus হিসেবে পরিচিতি লাভ করে। বর্তমানে হাজার হাজার নতুন Virus ও Malware তৈরি হচ্ছে।

⭐ Computer Virus-এর বৈশিষ্ট্য

  • 🦠 নিজের কপি তৈরি করতে পারে।
  • 📂 অন্যান্য ফাইলে সংক্রমণ ঘটায়।
  • ⚠️ সিস্টেম ধীর করে দেয়।
  • 🔒 তথ্য নষ্ট বা চুরি করতে পারে।
  • 🌐 USB, Email ও Internet-এর মাধ্যমে ছড়ায়।
  • 🛡 Antivirus দ্বারা শনাক্ত করা সম্ভব।

⚙️ Computer Virus কীভাবে কাজ করে?

  1. 📥 প্রথমে Virus কম্পিউটারে প্রবেশ করে।
  2. 📂 কোনো File বা Program-এর সাথে নিজেকে যুক্ত করে।
  3. 🦠 নিজের কপি তৈরি করে অন্যান্য File-এ ছড়িয়ে পড়ে।
  4. ⚠️ System-এর ক্ষতি করা শুরু করে।
  5. 📤 USB, Email বা Internet-এর মাধ্যমে অন্য কম্পিউটারে ছড়িয়ে যায়।

🦠 Computer Virus-এর প্রকারভেদ

Virus কাজ
Boot Sector Virus Boot Sector আক্রান্ত করে
File Infector Virus Program File আক্রান্ত করে
Macro Virus Word ও Excel Document আক্রান্ত করে
Resident Virus Memory-তে থেকে কাজ করে
Multipartite Virus Boot Sector ও File উভয় আক্রান্ত করে
Polymorphic Virus নিজের Code পরিবর্তন করে Antivirus এড়ায়

💾 Boot Sector Virus

এই Virus Hard Disk-এর Boot Sector আক্রান্ত করে। ফলে কম্পিউটার চালু হতে সমস্যা হয় এবং অনেক সময় Operating System Boot হয় না।

📂 File Infector Virus

এই Virus বিভিন্ন Program File (.exe, .com) আক্রান্ত করে এবং Program চালু হওয়ার সাথে সাথে সক্রিয় হয়ে যায়।

📄 Macro Virus

Microsoft Word, Excel বা অন্যান্য Document-এর Macro ব্যবহার করে ছড়িয়ে পড়ে।

🧠 Resident Virus

Resident Virus RAM-এ অবস্থান করে এবং Background-এ কাজ করতে থাকে। এটি নতুন File তৈরি বা খোলার সময় সংক্রমণ ঘটাতে পারে।

🛡️ Multipartite Virus

এই Virus একই সঙ্গে Boot Sector এবং Program File উভয়কেই আক্রান্ত করতে পারে। তাই এটি তুলনামূলক বেশি বিপজ্জনক।

🎭 Polymorphic Virus

Polymorphic Virus প্রতিবার সংক্রমণের সময় নিজের Code পরিবর্তন করে। ফলে Antivirus-এর জন্য এটি শনাক্ত করা কঠিন হয়ে যায়।

⚠️ Computer Virus-এর লক্ষণ

  • 🐢 কম্পিউটার ধীরে কাজ করে।
  • 💥 বারবার Hang বা Restart হয়।
  • ❌ File নিজে থেকেই Delete বা Corrupt হয়ে যায়।
  • 📂 অজানা File ও Folder তৈরি হয়।
  • 📢 অপ্রয়োজনীয় Pop-up Advertisement দেখা যায়।
  • 🌐 Internet অস্বাভাবিকভাবে Slow হয়ে যায়।
  • ⚙️ কিছু Program খুলতে সমস্যা হয়।
  • 🔋 CPU ও RAM-এর ব্যবহার অস্বাভাবিক বেড়ে যায়।

💥 Computer Virus-এর ক্ষতি

  • 📂 গুরুত্বপূর্ণ Data নষ্ট করে।
  • 🔒 File Encrypt করে মুক্তিপণ (Ransom) দাবি করতে পারে।
  • 👤 ব্যক্তিগত তথ্য ও Password চুরি করতে পারে।
  • 💻 Operating System ক্ষতিগ্রস্ত হতে পারে।
  • 🌐 Network-এর অন্যান্য Computer-এ ছড়িয়ে পড়তে পারে।
  • 💸 আর্থিক ক্ষতির কারণ হতে পারে।

📥 Virus কীভাবে ছড়ায়?

  • 💾 Infected USB Drive
  • 📧 Email Attachment
  • 🌐 ক্ষতিকর Website
  • ⬇️ Pirated Software Download
  • 🎮 Crack Software ও Game
  • 📁 File Sharing
  • ☁️ অনিরাপদ Download

🛡️ Virus থেকে বাঁচার উপায়

  • ✅ সবসময় Updated Antivirus ব্যবহার করুন।
  • ✅ Windows ও Software Update রাখুন।
  • ✅ অজানা Email Attachment খুলবেন না।
  • ✅ Pirated Software ব্যবহার করবেন না।
  • ✅ USB ব্যবহারের আগে Scan করুন।
  • ✅ গুরুত্বপূর্ণ Data Backup রাখুন।
  • ✅ Firewall চালু রাখুন।
  • ✅ নিরাপদ Website থেকে Download করুন।

🧹 Virus অপসারণের উপায়

  1. 🛡️ Antivirus দিয়ে Full Scan করুন।
  2. 🔄 Virus Definition Update করুন।
  3. 🗑️ আক্রান্ত File Delete বা Quarantine করুন।
  4. 💾 প্রয়োজনে Safe Mode-এ Scan করুন।
  5. ♻️ গুরুতর সংক্রমণে Operating System পুনরায় Install করুন।

📌 গুরুত্বপূর্ণ তথ্য

  • 🦠 সব Malware Virus নয়, তবে সব Virus হলো Malware।
  • 🛡️ Antivirus ১০০% নিরাপত্তা দিতে পারে না।
  • 🔄 নিয়মিত Update করলে নিরাপত্তা অনেক বাড়ে।
  • 👨‍💻 নিরাপদ Browsing অভ্যাস Virus প্রতিরোধে গুরুত্বপূর্ণ।

❓ FAQ

১. Computer Virus কী?

Computer Virus হলো এমন একটি ক্ষতিকারক Program যা কম্পিউটারের ক্ষতি করে এবং অন্য ফাইলে ছড়িয়ে পড়ে।


২. Virus কীভাবে ছড়ায়?

USB Drive, Internet, Email Attachment, Pirated Software এবং ক্ষতিকর Website-এর মাধ্যমে।


৩. Antivirus কি সব Virus ধরতে পারে?

বেশিরভাগ Virus ধরতে পারে, তবে Updated না থাকলে নতুন Virus শনাক্ত নাও করতে পারে।


৪. Virus থেকে বাঁচার সবচেয়ে ভালো উপায় কী?

Updated Antivirus ব্যবহার করা এবং সন্দেহজনক File Download না করা।
