📽️ MS PowerPoint কী?

Microsoft PowerPoint হলো Microsoft-এর তৈরি একটি জনপ্রিয় Presentation Software। এটি দিয়ে Slide তৈরি করে পাঠদান, মিটিং, সেমিনার এবং বিভিন্ন বিষয় সুন্দরভাবে উপস্থাপন করা যায়।

📄 Presentation ও Slide

Presentation: একাধিক Slide-এর সমষ্টিকে Presentation বলা হয়।

Slide: Presentation-এর প্রতিটি পৃথক পৃষ্ঠাকে Slide বলা হয়।

➕ নতুন Slide যোগ করা

নতুন Slide যোগ করতে:

Home → New Slide

Shortcut Key: Ctrl + M

✍️ Text যোগ করা

Slide-এর Text Box-এ ক্লিক করে লেখা টাইপ করা যায়।

Font, Size, Color এবং Alignment পরিবর্তন করা যায়।

🖼️ Picture যোগ করা

ছবি যোগ করতে:

Insert → Pictures

তারপর কম্পিউটার থেকে ছবি নির্বাচন করে Insert চাপুন।

🔷 Shape যোগ করা

Shape যোগ করতে:

Insert → Shapes

Arrow, Rectangle, Circle, Line ইত্যাদি ব্যবহার করা যায়।

💡 পরীক্ষার জন্য গুরুত্বপূর্ণ তথ্য

  • ✅ MS PowerPoint একটি Presentation Software।
  • ✅ একাধিক Slide নিয়ে একটি Presentation তৈরি হয়।
  • ✅ Ctrl + M = New Slide।
  • ✅ Picture যোগ করা হয় Insert → Pictures থেকে।
  • ✅ Shape যোগ করা হয় Insert → Shapes থেকে।
  • ✅ PowerPoint শিক্ষা, ব্যবসা ও সেমিনারে ব্যাপকভাবে ব্যবহৃত হয়।

🎬 Animation

Animation ব্যবহার করে Slide-এর লেখা, ছবি বা Shape-এ বিভিন্ন ধরনের চলমান (Motion) প্রভাব যোগ করা যায়।

ব্যবহার করতে:
Animations → Animation নির্বাচন করুন

🔄 Transition

Transition হলো এক Slide থেকে অন্য Slide-এ যাওয়ার সময় যে Effect দেখা যায়।

ব্যবহার করতে:
Transitions → Transition নির্বাচন করুন

🎤 Slide Show

Presentation প্রদর্শনের জন্য Slide Show ব্যবহার করা হয়।

Shortcut Key: F5

বর্তমান Slide থেকে শুরু করতে:
Shift + F5

💾 Presentation Save

Presentation সংরক্ষণ করতে:

Ctrl + S

নতুন Presentation তৈরি করতে:
Ctrl + N

❓ MS PowerPoint MCQ

  1. MS PowerPoint কী ধরনের Software?
    A) Spreadsheet Software
    B) Presentation Software
    C) Browser
    D) Operating System
    উত্তর: B) Presentation Software

  2. একাধিক Slide-এর সমষ্টিকে কী বলা হয়?
    A) Workbook
    B) Presentation
    C) Folder
    D) Database
    উত্তর: B) Presentation

  3. নতুন Slide যোগ করার Shortcut Key কী?
    A) Ctrl + M
    B) Ctrl + N
    C) Ctrl + S
    D) Ctrl + P
    উত্তর: A) Ctrl + M

  4. Picture কোথা থেকে যোগ করা হয়?
    A) Home
    B) Insert
    C) View
    D) Design
    উত্তর: B) Insert

  5. Shape কোথা থেকে যোগ করা হয়?
    A) Insert → Shapes
    B) Home → Shapes
    C) Review → Shapes
    D) View → Shapes
    উত্তর: A) Insert → Shapes

  6. Slide Show শুরু করার Shortcut Key কী?
    A) Ctrl + P
    B) F5
    C) Ctrl + O
    D) Alt + F4
    উত্তর: B) F5

  7. বর্তমান Slide থেকে Slide Show শুরু করতে কোন Shortcut ব্যবহার করা হয়?
    A) Ctrl + F5
    B) Shift + F5
    C) Alt + F5
    D) Ctrl + Shift + F5
    উত্তর: B) Shift + F5

  8. Animation কী কাজে ব্যবহৃত হয়?
    A) Slide-এ চলমান Effect যোগ করতে
    B) হিসাব করতে
    C) Internet চালাতে
    D) Print করতে
    উত্তর: A) Slide-এ চলমান Effect যোগ করতে

  9. Transition কী?
    A) দুই Slide-এর মধ্যে পরিবর্তনের Effect

    C) ছবি যোগ করা
    D) Table তৈরি
    উত্তর: A) দুই Slide-এর মধ্যে পরিবর্তনের Effect

  10. Presentation Save করার Shortcut Key কী?
    A) Ctrl + O
    B) Ctrl + S
    C) Ctrl + M
    D) Ctrl + T
    উত্তর: B) Ctrl + S
