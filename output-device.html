🖥️ Output Devices

Learning Skills

🖥️ Monitor

Monitor হলো সবচেয়ে গুরুত্বপূর্ণ Output Device। এটি কম্পিউটারের তথ্য, ছবি ও ভিডিও আমাদের দেখায়।

🖨️ Printer

Printer কম্পিউটারের Soft Copy-কে কাগজে Print করে Hard Copy তৈরি করে।

🔊 Speaker

Speaker-এর মাধ্যমে কম্পিউটার থেকে শব্দ (Sound) শোনা যায়।

🎧 Headphone

Headphone ব্যবহার করে ব্যক্তিগতভাবে গান, ভিডিও বা অন্যান্য Audio শোনা যায়।

📽️ Projector

Projector কম্পিউটারের ছবি বা ভিডিও বড় পর্দায় প্রদর্শন করে। এটি স্কুল, কলেজ ও অফিসে Presentation-এর জন্য ব্যবহৃত হয়।

📺 Output Device-এর উদাহরণ

Device কাজ
Monitor তথ্য প্রদর্শন
Printer Print করা
Speaker শব্দ শোনানো
Projector বড় পর্দায় প্রদর্শন

💡 পরীক্ষার জন্য গুরুত্বপূর্ণ তথ্য

  • ✅ Monitor একটি Output Device।
  • ✅ Printer Hard Copy তৈরি করে।
  • ✅ Speaker শব্দ Output দেয়।
  • ✅ Projector Presentation-এর জন্য ব্যবহৃত হয়।

❓ Output Device MCQ

  1. Output Device কোনটি?
    A) Keyboard
    B) Mouse
    C) Monitor
    D) Scanner
    উত্তর: C) Monitor

  2. Printer কী তৈরি করে?
    A) Soft Copy
    B) Hard Copy
    C) Program
    D) Database
    উত্তর: B) Hard Copy

  3. শব্দ শোনার জন্য কোন Device ব্যবহৃত হয়?
    A) Speaker
    B) Keyboard
    C) Mouse
    D) Scanner
    উত্তর: A) Speaker

  4. Projector কোথায় বেশি ব্যবহৃত হয়?
    A) Presentation
    B) Typing
    C) Scanning
    D) Gaming
    উত্তর: A) Presentation

  5. Headphone কী ধরনের Device?
    A) Input
    B) Output
    C) Storage
    D) Processing
    উত্তর: B) Output
