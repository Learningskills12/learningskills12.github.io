🖥️ Hardware কী?

Hardware হলো কম্পিউটারের সকল দৃশ্যমান (Physical) অংশ, যেগুলো আমরা দেখতে ও স্পর্শ করতে পারি। যেমন: Monitor, Keyboard, Mouse, CPU Cabinet, Printer ইত্যাদি।

🔧 Internal Hardware

Internal Hardware হলো কম্পিউটারের ভিতরে থাকা যন্ত্রাংশ।

  • 🧠 CPU
  • 💾 RAM
  • 💿 ROM
  • 💽 Hard Disk / SSD
  • 🖥️ Motherboard
  • 🔋 SMPS (Power Supply)

🖱️ External Hardware

External Hardware হলো কম্পিউটারের বাইরে সংযুক্ত যন্ত্রাংশ।

  • ⌨️ Keyboard
  • 🖱️ Mouse
  • 🖥️ Monitor
  • 🖨️ Printer
  • 🔊 Speaker
  • 🎤 Microphone
  • 📷 Webcam

📦 Hardware-এর প্রকারভেদ

ধরন উদাহরণ
Input Device Keyboard, Mouse, Scanner
Output Device Monitor, Printer, Speaker
Storage Device Hard Disk, SSD, Pen Drive
Processing Device CPU

💡 পরীক্ষার জন্য গুরুত্বপূর্ণ তথ্য

  • ✅ Hardware দেখা ও স্পর্শ করা যায়।
  • ✅ CPU একটি Processing Hardware।
  • ✅ Keyboard একটি Input Hardware।
  • ✅ Monitor একটি Output Hardware।
  • ✅ Hard Disk একটি Storage Hardware।

❓ Hardware MCQ

  1. Hardware কী?
    A) দেখা যায় না
    B) স্পর্শ করা যায় না
    C) কম্পিউটারের দৃশ্যমান অংশ
    D) শুধুমাত্র Program
    উত্তর: C) কম্পিউটারের দৃশ্যমান অংশ

  2. নিচের কোনটি Hardware?
    A) Windows
    B) MS Word
    C) Keyboard
    D) Chrome
    উত্তর: C) Keyboard

  3. CPU কী ধরনের Hardware?
    A) Input
    B) Output
    C) Processing
    D) Storage
    উত্তর: C) Processing

  4. Hard Disk কী?
    A) Storage Device
    B) Input Device
    C) Output Device
    D) Software
    উত্তর: A) Storage Device

  5. নিচের কোনটি Input Device?
    A) Monitor
    B) Printer
    C) Mouse
    D) Speaker
    উত্তর: C) Mouse

  6. নিচের কোনটি Output Device?
    A) Scanner
    B) Keyboard
    C) Monitor
    D) Microphone
    উত্তর: C) Monitor

  7. RAM কোথায় থাকে?
    A) কম্পিউটারের ভিতরে
    B) Monitor-এর ভিতরে
    C) Printer-এর ভিতরে
    D) Keyboard-এর ভিতরে
    উত্তর: A) কম্পিউটারের ভিতরে

  8. Motherboard কী?
    A) Software
    B) Hardware
    C) Browser
    D) Operating System
    উত্তর: B) Hardware

  9. Speaker কোন ধরনের Hardware?
    A) Input
    B) Storage
    C) Output
    D) Processing
    উত্তর: C) Output

  10. Hardware-এর উদাহরণ কোনটি?
    A) MS Excel
    B) Windows
    C) Printer
    D) Google Chrome
    উত্তর: C) Printer
